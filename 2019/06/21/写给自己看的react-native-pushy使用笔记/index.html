语义化版本规范

  1. 修改你的业务代码

  2. pushy bundle --platform android, 之后输入Y立即发布（此过程是生成新的热更新版本）

  3. 此时版本已经提交到update服务，但用户暂时看不到此更新，你还需要将此此热更新版本绑定到指定的包版本上。
    有3种方法：

    • 输入Y立即绑定
    • 或者以后pushy update --platform android来让对应包版本用户得以更新
      • 输入Y或者上述命令以后，会让你再输入热更新版本号versionId，就是)前面的数字
      • （可以使用U上一页，D下一页，B回到开始）
      • 输入完versionId以后，会让你输入包版本号packageId，就是U/D/B提示下面的列表前的数字
    • 还可以直接在网页端拖拽操作

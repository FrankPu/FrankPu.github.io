布局：水平居中、垂直居中、水平垂直居中

水平垂直居中

flex + margin（非常推荐）

flex + justify-content + align-items（推荐）

absolute + transform（推荐）

inline-block + text-align + table-cell + vertical-align

水平居中

absolute + transform（推荐）

transform为css3属性，兼容到IE10，IE9添加-ms-transform 属性

flex + margin（推荐）

flex为css3属性，兼容到IE9

flex + justify-content（推荐）

flex为css3属性，兼容到IE9

inline-block + text-align

把子元素设为inline-block 元素，然后对父元素设置text-align:center 让子元素居中，兼容到IE6

table + margin

兼容到IE8

垂直居中

absolute + transform（推荐）

类似于水平居中的那个方法

transform为css3属性，兼容到IE10，IE9添加-ms-transform 属性

flex + align-items（推荐）

类似于水平居中的那个方法

flex为css3属性，兼容到IE9

table-cell + vertical-align

兼容到IE8
